AaData

Asistente de análisis de datos paso a paso para investigadores sin experiencia en programación.

Ver en GitHub Descargar (.zip) ← Volver a Proyectos

Descripción

AaData es una aplicación web construida en R con Shiny que guía al investigador, pregunta tras pregunta, a lo largo de un asistente de siete pasos —desde cargar un archivo hasta descargar un reporte— para limpiar, analizar y graficar sus datos. Está orientada a investigación clínica y de ciencias básicas, con ejemplos pensados para un modelo murino de artritis reumatoide.

Funcionalidades principales

Asistente por pasos

Flujo guiado de 7 pasos con barra de progreso: bienvenida, carga, vista previa, limpieza, análisis, gráficas y descargas.

Carga flexible

Datos de ejemplo, archivos CSV/Excel e importación avanzada de Excel con encabezados en varias filas y celdas combinadas.

Limpieza de datos

Faltantes, duplicados, estandarización de texto, conversión numérica, redondeo, filtro por rango, detección de atípicos y más.

Análisis estadístico

Descriptivos, comparación de grupos, pareado, correlaciones, ANOVA de dos vías, modelo mixto, regresiones y normalidad.

Gráficas

Histograma, boxplot, violín, dispersión, barras con error estándar, mapa de calor, línea de tiempo y Kaplan-Meier.

Reportes

Exportación de datos a CSV/Excel y un reporte completo en PDF con resultados, gráficas y bitácora de la sesión.

Tecnologías

R Shiny ggplot2 dplyr tidyr readxl survival nlme gridExtra

Aspectos destacados

  • Resultados explicados en lenguaje sencillo, no solo el valor p.
  • Selección automática de pruebas paramétricas o no paramétricas según la normalidad de los datos.
  • Manejo robusto de errores para que la sesión no se interrumpa ante datos inesperados.
  • Todo el procesamiento ocurre localmente en la computadora del investigador (sin enviar datos a servidores externos).
